Workflow Recomendado

1. Verificar Backend

Asegúrate que el servidor backend está corriendo. Usa "Verificar API" para comprobar.

2. Abrir Panel Admin

El panel se auto-autentica. Verifica el dashboard y crea una operación con imagen.

3. Ver en Frontend

Abre el Frontend en otra pestaña y verifica que la operación aparece con imagen.

4. Probar Login

En Frontend, haz click en "Iniciar Sesión" y prueba con ambos usuarios.

5. Probar CRUD

Desde Admin Panel, edita y elimina operaciones. Verifica cambios en tiempo real.
